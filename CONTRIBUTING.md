# 贡献说明

欢迎改进这个财务自由工作年限计算器。

## 建议的贡献方向

- 改进计算模型
- 增加更多可选人生事件
- 优化移动端体验
- 增加图表可读性
- 修复边界参数下的计算问题
- 改进 README 或示例说明

## 修改原则

- 保持项目为纯静态网页
- 不引入不必要的构建工具
- 不上传个人财务数据
- 新增参数时，尽量说明它如何影响现金流或退休门槛
- 涉及默认值时，避免把某种人生选择设为必须假设

## 本地检查

修改后建议至少检查：

```bash
node --check script.js
```

然后在浏览器中打开 `index.html`，确认页面可以正常加载、输入参数可以实时更新结果。
