# 财务自由工作年限计算器

一个纯前端静态网页工具，用来估算一个人需要工作多久，才能在当前假设下覆盖一生的财务目标。

项目不需要后端、不需要数据库、不需要构建工具。把 `index.html`、`styles.css`、`script.js` 放在同一个目录下，直接打开 `index.html` 就能使用。

## 功能

- 根据工作起始年龄、预计寿命、月收入、月开销、起始资产和年化收益率计算所需工作年限
- 支持工作期收益率和退休后收益率分别设置
- 支持分段收入，到达某个年龄后可调整个人每月税后收入
- 支持退休后被动收入、退休时一次性目标、去世时保留资产和安全垫
- 支持可选人生事件：
  - 买房置业，包括贷款总额、贷款利率、每月房贷和房贷年限联动计算
  - 结婚与家庭，包括婚礼安家支出、伴侣分担的月收入和家庭每月额外开销
  - 生娃养育，可分别配置每个孩子的出生年龄、教育储备和彩礼/婚嫁支持
  - 父母赡养/照护
  - 养老储备金和养老额外每月预算
- 每个可选事件都可以单独开启或关闭
- 自动记住上次输入的参数，下次打开会恢复
- 支持保存多个快速设置方案，并在页面顶部快速切换
- 重要假设提醒可以关闭，关闭后下次打开不会再次显示
- 当任意时刻资产小于 0 时提示破产风险
- 使用资产轨迹图展示资产变化和退休门槛
- 展示关键人生节点时间线
- 支持生成可预览、可下载的 PNG 图片看板

## 本地使用

直接双击打开：

```text
index.html
```

保持下面三个核心文件在同一个文件夹中：

```text
index.html
styles.css
script.js
```

## 发布到 GitHub Pages

这个项目是静态网页，适合直接使用 GitHub Pages 发布。

1. 新建一个 GitHub 仓库
2. 上传本项目文件
3. 在仓库中进入 `Settings` -> `Pages`
4. `Source` 选择 `Deploy from a branch`
5. 分支选择 `main`，目录选择 `/ (root)`
6. 保存后等待 GitHub Pages 生成访问地址

## 计算假设

当前模型为了保持清晰和可理解，采用以下假设：

- 不考虑通货膨胀
- 不考虑税费变化
- 未设置分段收入时，个人收入按起始月收入保持不变
- 不考虑职业中断
- 不考虑投资收益波动，按输入的年化收益率折算为月收益率
- 生活开销、事件开销、父母赡养和养老储备会按安全垫放大
- 可选人生事件关闭后，不参与现金流、退休门槛和时间线计算

计算结果只适合做个人规划参考，不构成投资、税务、法律或保险建议。

## 开发

项目没有依赖。修改后可以直接刷新浏览器查看效果。

如果想启动一个本地静态服务，也可以在项目目录中运行：

```bash
python -m http.server 4174
```

然后访问：

```text
http://127.0.0.1:4174/index.html
```
